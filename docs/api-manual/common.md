## 流式处理（Streaming）

流式处理是一种逐步返回数据的方式，适用于需要快速反馈和更流畅用户体验的场景。

### 特点
- 逐步返回响应：模型会在生成内容时实时返回数据，而不是等到整个回答完成后再一次性返回。
- 更快的用户体验：用户可以更快地看到部分响应，而无需等待完整结果。
- 适用于实时交互：特别适用于聊天机器人、实时翻译等应用场景。
- 减少超时风险：对于长文本输出，流式处理可以降低因超时导致的请求失败。
